INTRODUCTION
---------------------

SimpleBlog is a new and clean, grid responsive Drupal theme on the web.
This theme can be used for drupal to create your powerful Blogging website
with grid layout.

Following are the featrues for the theme:

 * Responsive Blog
 * Simple clean design
 * Grid Layout (Masonry grid layout library)
 * With/without Grid Layout using Configuration
 * Supports 3-columns layout
 * Grid layout Supports only for Views front page & taxonomy pages

 For a full description of the theme, visit the project page:
   https://www.drupal.org/project/simpleblog

 To submit bug reports and feature suggestions, or to track changes:
   https://www.drupal.org/project/issues/simpleblog


REQUIREMENTS
---------------------

No special requirements.


INSTALLATION
---------------------

Install as you would  normally install a Druapl contributed theme.
Visit: https://www.drupal.org/docs/extending-drupal/installing-themes for
further informtaion.


CONFIGURATION
---------------------

Configure theme settings in Administration » Appearance » Settings » Simpleblog:

Under SIMPLEBLOG Settings:

  * Check to show grid view i9n the home page.

  * For Banner
    - Check "Enable banner" if want to show banner on pages.
    - Can configure to show banner on home page.

  * Copyright
    - Enter the copyright text you want to show in the footer.

  * Social Icons:
    - Enter the links of the social icons you want to show in the footer of your
      website.
